# MVP Scope

## In Scope

- Account registration and login
- Onboarding for initial taste profile (favorite/disliked genres, favorite titles)
- Search for movies/TV series
- Title details page
- Expectation scoring for unwatched titles (0-10)
- Watched state tracking
- Post-watch rating (0-10) after watched confirmation
- Delta computation per title (`expectation - rating`)
- Basic personalized recommendation feed using recency + rating + delta signals
- Availability display by region with freshness disclaimer

## Out of Scope (MVP)

- Native mobile apps
- Advanced social graph features (comments/likes/feed mechanics)
- Real-time watch parties
- Paid subscriptions
- Full advanced analytics dashboard

## MVP Success Metrics

- Median time-to-decision before first play
- Weekly active raters
- Expectation-to-rating completion rate
- Recommendation acceptance rate (title started after recommendation)
- Home page p75 load time on mobile
