# MVP Scope

## In Scope

- Account registration and login
- User profile
- Search for movies/TV series
- Title details page
- Rate title (1-10)
- Basic text review
- Follow/unfollow users
- Activity feed from followed users
- Initial recommendation feed

## Out of Scope (MVP)

- Native mobile apps
- Advanced moderation automation
- Real-time watch parties
- Paid subscriptions

## MVP Success Metrics

- Weekly active users
- Ratings per active user
- Follow graph growth
- Recommendation click-through rate
